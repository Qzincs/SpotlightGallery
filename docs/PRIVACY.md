# 隐私政策

**最后更新日期：** 2025年11月4日

欢迎您使用由 Qzink（以下简称“我们”）开发的 SpotlightGallery 应用程序（以下简称“本程序”）。我们高度重视您的隐私权，并致力于保护您的个人信息。

本隐私政策旨在向您说明，本程序**不会**收集您的任何信息。

### 1. 信息的收集与使用

我们在此明确声明：

**SpotlightGallery 应用程序本身不会以任何方式主动收集、存储、传输或共享您的任何个人信息或匿名使用数据。**

* 本程序不包含任何形式的用户账户系统。
* 本程序不使用任何第三方分析工具（如 Google Analytics 等）。
* 本程序不投放任何广告，也不包含任何广告 SDK。
* 您的所有设置和偏好仅存储在您本地的设备上，我们无法访问。

### 2. 用户自愿提供的信息

本程序在 GitHub 上开源。我们唯一可能接触到您信息途径是：当您在使用过程中遇到程序错误（Bug）并**主动选择**通过 GitHub 平台提交 "Issue"（问题报告）时。

* **日志信息：** 为了帮助我们诊断并解决问题，您**可能会主动选择**复制并粘贴本程序的运行日志。这些日志可能包含有关您设备配置（如操作系统版本）和程序错误堆栈的技术信息。
* **使用目的：** 您在 GitHub 上提供的任何信息（包括日志）将**仅用于**诊断和解决您所报告的特定技术问题。我们不会将此信息用于任何其他目的，例如用户分析、营销或身份识别。

我们建议您在提交日志之前，自行审阅日志内容，以确保没有包含您不希望公开的任何敏感个人信息。

### 3. 数据的安全性

由于本程序不收集您的任何数据，因此不存在数据泄露的风险。对于您自愿在 GitHub 上提交的数据，我们将依赖 GitHub 平台提供的安全措施。

### 4. 本隐私政策的变更

我们可能会不时更新本隐私政策。任何变更都将发布在此页面上。由于本程序不收集您的联系方式，我们无法主动通知您。建议您定期查看本政策以了解任何更新。
